Instruction to the laboratory

  1. Study the theory
  2. Select the height of tank, substance type and size of radius and submit the values.
  3. First is characteristics of C =f(substance) for different substance level.
    1. Select the percentage representation of substance that will fill the container.
    2. Use cylindrical capacitance form to calculate the output capacitance and fill the gap with this value (capacitance should be in [nF] with accuracy to 2 decimal numbers.
    3. If the value is wrong, try again. The value is correct when the tank fill with the substance. Add other points to the characteristic.
    4. To see the graph, click “See graph” button or change the substance with button “Change the substance”.
  4. Go to characteristic C = f(r1) for different substance level, by clicking the link on the top of the website.
    1. First select the radius size and submit.
    2. Next do the same step from 3. a) to d) and see the results on graph.
  5. Go to the characteristic C =f(T) for different substances, by clicking the link on the top of the website.
    1. First choose the substance and submit.
    2. Select the temperature.
    3. Find in table the value of relative dielectric permittivity of this substance for selected temperature and repeat the steps from 3. b) to d).
